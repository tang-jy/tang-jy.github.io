lambda在 C++11中，引入了 lambda 表达式，它允许我们在需要函数对象的地方编写匿名函数。Lambda 表达式的基本语法如下： 123[capture](parameters) -> return_type { // 函数体} 其中： capture：捕获列表，用于捕获外部变量，可以是空的[]，也可以是[var]或[var1, var2]等形式。 parameters：参数列表，与普通函数的参数列表类似。 return_type：返回类型，可以省略，根据函数体自动推断返回类型。 {} 内为函数体。
